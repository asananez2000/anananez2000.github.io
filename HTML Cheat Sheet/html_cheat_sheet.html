Tags Básicos

Antes que nada, es importante mencionar que un "tag" es una parte del markup language que indica en inicio y el final de un elemento de HTML. A continuación se utilizan y describen los tags básicos al momento de escribir HTML.

< html > < /html > : Tag que crea un documento HTML.

< head > < /head > : Tag que contiene los metadatos. Información que necesita ser procesada por la computadora y no se muestra en el documento (title, style sheets etc.).

< body > < /body > : Tag que contiene la todos los elementos visibles del documento.

< title > < /title >: Tag que pone el nombre del documento en la barra de título.

< br > : Tag que produce un salto de línea.

Dentro del Div

El tag < div > < /div > es un elemento que sirve como un contenedor genérico que por ende no tiene ningún efecto en el estilo de su contenido. Es útil para dar estructura en la página y al momento de dar estilo a cierto contenido

Toda la sección "Dentro del Div" está dentro de un div


Tags Semánticos

Los tags semánticos son aquellos elementos en HTML cuyo nombre describe de forma clara su función. Por ejemplo el nombre del elemento < table > < /table > decribe con claridad que sirve para crear una tabla.


Dentro del Header

Los headers < header > < /header > son un tag semántico que especifican el encabezado de alguna sección. Se utilizan como contenedores para contenido introductorio. Es como un < div > glorificado, el cual semanticamente (tanto para el buscador como para el desarrollador) el propósito de agrupar dicho tipo de contenido.

Toda la sección "Dentro del Header" está dentro de un header.


Dentro del Span

El tag < span > < /span > es un elemento que crea un contenedor en línea por lo que es útil para aplicar estilo a parte de un texto o incluso parte de un documento.

Dentro de Article

El tag < article > < /article > se utiliza comunmente para contener información que tiene sentido por sí misma y que puede distribuirse independientement del sitio en el que está. Ej: blogs o noticias.

Listas

En HTML existen ciertos elementos para crear listas. Especificamente hay dos tipos, las listas no ordenadas y las listas ordenadas

Listas no Ordenadas (unordered lists)

Se utiliza el tag < ul > < /ul > que crea una lista no ordenada. Dentro de este elemento se utiliza el tag < li > < /li > (list element) para agregar cada elemento de la lista. Como este tag está dentro de un elemento < ul > < /ul > el texto se marca con viñetas.

  • Elemento 1
  • Elemento 2
  • Elemento 3

Lista Ordenadas (ordered lists)

Se utiliza el tag < ol > < /ol > que crea una lista ordenada. Dentro de este elemento se utiliza el tag < li > < /li > (list element) para agregar cada elemento de la lista. Como este tag está dentro de un elemento < ol > < /ol > el texto se marca con enumeración.

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Tablas

En HTML también existen elementos para crear tablas. Estos de describen a continuación.

< table > < /table > : Tag que sirve para crear una tabla.

< thead > < /thead > : Tag que contiene el conjunto de filas que definen los encabezados de las columnas

< tbody > < /tbody > : Tag que tiene el contenido de la tabla encapsulando el conjunto de filas < tr > que la componen.

< tr > < /tr > : Tag que crea una fila en la tabla y en su interior contiene el conjunto de celdas < td > que la conforman.

< td > < /td > : Tag que crea una celda y define la información que contine.

Imágenes

En HTML existen diferentes formas para insertar imágenes en un documento. Todas las que se utilizan en este documento utilizan el elemento < img > que sirve para incorporar una imagen en el mismo.

src es un atributo del tag mencionado anteriormente, el cual contiene el "path" a la imagen que se quiere insertar. Este "path" puede ser relativo (foto de saturno) o puede ser una liga de internet (picsum e imagen de helados).

alt es un atributo del tag mencionado anteriormente, el cual contiene una breve descripción de la imagen que se muestra en caso de que esta no pueda ser cargada o mostrada.


Links

En HTML también se pueden crear ligas que conducen a otros documentos, para ellos se utilizan los siguientes elementos.

< a > : (anchor tag) Tag que por medio de su atributo href puede crear un hipervínculo a algún otro documento (pág de internet, archivo, email etc.). Este elemento también tiene el atributo target el cual sirve para indicar donde mostrar aquello a lo que lleva el hipervínculo.

Formas

Igualmente en HTML se pueden utilizar "formas" o contenedores que sirven para diferentes tipos de elmentos para que el usuario ingrese información (input elements). Posteriormente, se describen y ejemplifican los tags que sirven para lo anterior.

< form > < /form > : Tag que crea una forma y contiene sus diferentes elementos que la conforman.

< textarea > < /textarea > : Tag que crea un cuadro de texto cuyo tamaño o logitud del texto que sorporta es flexible (bueno para comentarios, reseñas etc.). Tiene atributo placeholder el cual sirve para dar una breve descripción al usuario.

< button > < /button > : Tag que crea un botón con su respectivo texto y la capacidad de interactuar con el mouse de usuario.

< input >: Tag que crea crea un cuadro de texto y es especialmente útil ya que permite una variedad de "input types" mediante su atributo type.

Name

atributo type="email": Atributo que hace que el cuadro de texto tenga parámetros de validación útiles para email.

Email

atributo type="password": Atributo que le da la funcionalidad necesaria para passwords al cuadro de texto (obscurece el texto y tiene botón de mostrar).

Password

atributo placeholder="Escribe aquí": Imput también tiene este atributo y sirve para dar una breve descripción acerca de lo que se debe de escribir.
