Font-family

Es una propiedad en CSS que define una lista priorizada con estilos de fuente (arial, calibri etc,). De esta lista se elige la primera fuente y las siguientes se vuelven alternativas que se utilizan en caso de que la fuente no se encuentre disponible. Por esto siempre es recomendable colocar una fuente genérica como sans-serif así asegurando que al menos hay una fuente disponible.

Ejemplo

En este ejemplo se cambió la fuente del texto a Times New Roman, y se colocaron como alternativas a la fuente Times, seguida de sans-serif.
